# INN

Versão Atual: v1.006

## Missão

Sistema operacional comercial com IA e voz.

## Regras Fundamentais

1. Um cliente = um cadastro.
2. Visitas não criam clientes.
3. Propostas não criam clientes.
4. Ligações não criam clientes.
5. Follow-up não cria clientes.
6. Tudo gera histórico.

## Prioridades

### P1

* Anti-duplicidade de clientes.
* Histórico de atividades.

### P2

* Voz conversacional.
* Parser inteligente.

### P3

* Timeline do cliente.
* Dashboard operacional.

## Problemas Conhecidos

* Parser de voz falha em alguns comandos.
* Possibilidade de duplicação de clientes.
* Histórico de atividades ainda não implementado.
